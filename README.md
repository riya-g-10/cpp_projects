## 🚀 Tasks Overview

### ✅ Task 1 — String Reversal
> Take a string as input and display it reversed.

- Reverses any user-inputted string
- Demonstrates string traversal using loops
- Core concept: **String manipulation**

---

### ✅ Task 2 — Calculator Program
> Take two numbers and an operator (`+`, `-`, `*`, `/`, `%`) and perform the arithmetic operation.

- Supports all basic arithmetic operations
- Handles edge cases like division by zero
- Core concept: **Conditional statements & I/O**

---

### ✅ Task 3 — Fibonacci Series
> Generate the Fibonacci series up to a user-specified number of terms.

- Prompts user for the number of terms
- Displays the complete series
- Core concept: **Loops & arithmetic logic**

---

### ✅ Task 4 — Guessing Game
> Generate a random number and let the user guess it with "higher/lower" hints.

- Random number generation using `rand()`
- Gives hints to guide the user
- Core concept: **Random numbers, loops & conditionals**

---

### ✅ Task 5 — File Copy
> Copy the contents of one file into another using file handling.

- Prompts user for source and destination filenames
- Reads and writes file contents
- Core concept: **File handling (`fstream`)**

---

### ✅ Task 6 — Palindrome Checker
> Check whether a given word or phrase reads the same forwards and backwards.

- Works for both words and full phrases
- Case-insensitive comparison
- Core concept: **String manipulation & conditionals**

---

### ✅ Task 7 — Student Grade Calculator
> Input grades for multiple subjects and calculate the average grade.

- Takes number of subjects as input
- Calculates and displays the average
- Core concept: **Arrays & arithmetic operations**

---

### ✅ Task 8 — Array Sorting
> Sort an array of integers in ascending or descending order.

- User chooses sorting direction
- Implements sorting algorithms (Bubble Sort / Selection Sort)
- Core concept: **Sorting algorithms & arrays**

---

### ✅ Task 9 — Rock-Paper-Scissors Game
> Play RPS against the computer with random choice generation.

- Computer makes a random choice each round
- Determines and displays the winner
- Core concept: **Random numbers & conditional logic**

---

### ✅ Task 10 — Inventory Management System
> Add items, display them, and search by name or ID.

- Supports add, display, and search operations
- Uses structs/classes for data modeling
- Core concept: **OOP, structs, arrays/linked lists**
